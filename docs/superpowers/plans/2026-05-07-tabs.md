# Tabs 组件 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `@component-ai/react-ui` 与 `@component-ai/vue-ui` 中实现与设计规格一致的复合组件 `Tabs` / `TabsList` / `TabsTrigger` / `TabsPanel`（横向 + 纵向、自动激活、键盘与 ARIA），并用 Vitest 做最小行为回归。

**Architecture:** React 侧使用单一上下文注册所有 Trigger 的 `value`、禁用态与插入顺序，在 Trigger 上处理箭头/Home/End 并驱动 `value` + `focus`；`TabsPanel` 用 `hidden` 隐藏非激活面板。Vue 侧使用 `provide`/`inject` + 相同注册模型，子组件为 TSX 与现有 vue-ui 一致。

**Tech Stack:** TypeScript、React 19、Vue 3.5、TSX（Vue）、Vite 8、Vitest、`@testing-library/react`、`@testing-library/user-event`、`@vue/test-utils`、Tailwind v4（`@tailwindcss/vite`）。

**规格来源:** [`docs/superpowers/specs/2026-05-07-tabs-design.md`](../specs/2026-05-07-tabs-design.md)

### Task 3: React Tabs — 实现（上下文 + 注册 + 样式 + 键盘）

**Files:**
- Create: `packages/react-ui/src/components/Tabs.tsx`
- Modify: `packages/react-ui/src/index.ts`

**实现要点（与规格对齐）：**

- `Tabs`：`orientation` 默认 `horizontal`；React 受控：`value !== undefined`；`onValueChange`；内部 state 仅在非受控时使用。
- 初次激活：非受控且无 `defaultValue` 时，在 Trigger `register` 之后（`useEffect`），取 **插入顺序** 中第一个 `disabled === false` 的 `value`。
- `TabsList`：`role="tablist"`，`aria-orientation`，默认 `aria-label="标签页"` 可被覆盖；`horizontal` 时 `flex flex-row gap-1 overflow-x-auto`，`vertical` 时 `flex flex-col gap-1`。
- 根节点：`orientation === 'vertical'` 时 `className` 含 `flex flex-row gap-4 items-start`，左侧 List、右侧为包裹所有 Panel 的容器（Panel 仍各自 `hidden`，可将 Panel 作为根的并列子节点：`<Tabs><TabsList/><div class="flex-1">{panels}</div></Tabs>` — **若结构调整会与测试 DOM 查询兼容**，保持每个 `TabsPanel` 仍为 `Tabs` 的直接子节点更简单：纵向时用根 `flex flex-row`，`TabsList` 不占满时用 `shrink-0`，_PANEL 包一层可选。
- **简化结构（推荐）**：`Tabs` 根仅 `div` + Provider；子节点顺序由使用者排列：`TabsList` 后并列多个 `TabsPanel`。纵向布局：`Tabs` 根 `flex flex-row gap-4`，`TabsList` `shrink-0`，用一个 `div.flex-1.flex.flex-col` 包裹 **所有** `TabsPanel` — 这需要约定使用者写法 OR Tabs 用 `children` 分拣——为降低复杂度：**纵向时要求使用者写成 `TabsList` + `div` 包裹 panels**，规格写的是「列表列 + 内容列」，计划在 API 文档注明：**纵向示例中 Panels 放在同一 `div` 内**。实现：`Tabs` 仅渲染 `children`，样式通过 `className` + 文档示例完成纵向排版。
- `TabsTrigger`：`role="tab"`，`id={baseId + '-tab-' + value}`，`aria-selected`，`tabIndex={选中 ? 0 : -1}`，`disabled` 时跳过键盘导航；`onClick` 调用 `setValue(value)`；`onFocus` 在自动激活模式下调用 `setValue(value)`；`onKeyDown` 处理箭头/Home/End（横向仅用 Left/Right，纵向仅用 Up/Down）。
- `TabsPanel`：`role="tabpanel"`，`id={baseId + '-panel-' + value}`，`aria-labelledby` 指向对应 tab id，`hidden={value !== active}`。
- **注册表**：Context 提供 `registerTab(value: string, disabled: boolean)`、`unregisterTab(value: string)`、`tabsOrderRef` 用 `useRef<string[]>` 维护插入顺序（Register 时若未包含则 push；unregister 时 filter）。
- **开发环境**：若同一 `value` 注册两次，`console.warn`。
- **键盘**：从当前 `value` 在 `tabsOrderRef` 中找索引，向 next/prev 寻找下一个非 `disabled`（disabled 状态存在 Context `disabledRef: Record<string, boolean>` 或 Map）。

由于篇幅，工程师可将下列代码整块写入 `Tabs.tsx` 并按 TypeScript 报错微调类型：

```tsx
import {
  createContext,
  useCallback,
  useContext,
  useEffect,
  useId,
  useMemo,
  useRef,
  useState,
  type KeyboardEvent,
  type ReactNode,
} from "react";

type Orientation = "horizontal" | "vertical";

type TabsContextValue = {
  baseId: string;
  orientation: Orientation;
  value: string;
  setValue: (next: string) => void;
  registerTab: (value: string, disabled: boolean) => void;
  unregisterTab: (value: string) => void;
  isDisabled: (value: string) => boolean;
  tabOrder: string[];
};

const TabsContext = createContext<TabsContextValue | null>(null);

function useTabsContext(component: string) {
  const ctx = useContext(TabsContext);
  if (!ctx) throw new Error(`${component} must be used inside <Tabs>`);
  return ctx;
}

export type TabsProps = {
  orientation?: Orientation;
  value?: string;
  defaultValue?: string;
  onValueChange?: (value: string) => void;
  className?: string;
  children?: ReactNode;
};

export function Tabs({
  orientation = "horizontal",
  value: valueProp,
  defaultValue,
  onValueChange,
  className = "",
  children,
}: TabsProps) {
  const baseId = useId().replace(/:/g, "");
  const isControlled = valueProp !== undefined;
  const [internalValue, setInternalValue] = useState<string | undefined>(
    defaultValue,
  );
  const selected = isControlled ? valueProp! : internalValue ?? "";

  const tabOrderRef = useRef<string[]>([]);
  const disabledRef = useRef<Record<string, boolean>>({});
  const [, bump] = useState(0);
  const rerender = () => bump((x) => x + 1);

  const registerTab = useCallback((value: string, disabled: boolean) => {
    if (
      import.meta.env?.DEV &&
      tabOrderRef.current.includes(value)
    ) {
      console.warn(`[Tabs] duplicate TabsTrigger value: ${value}`);
    }
    if (!tabOrderRef.current.includes(value)) tabOrderRef.current.push(value);
    disabledRef.current[value] = disabled;
    rerender();
  }, []);

  const unregisterTab = useCallback((value: string) => {
    tabOrderRef.current = tabOrderRef.current.filter((v) => v !== value);
    delete disabledRef.current[value];
    rerender();
  }, []);

  const setValue = useCallback(
    (next: string) => {
      if (disabledRef.current[next]) return;
      onValueChange?.(next);
      if (!isControlled) setInternalValue(next);
    },
    [isControlled, onValueChange],
  );

  useEffect(() => {
    if (isControlled) return;
    if (defaultValue !== undefined) return;
    if (internalValue !== undefined && internalValue !== "") return;
    const order = tabOrderRef.current;
    const first = order.find((v) => !disabledRef.current[v]);
    if (first !== undefined) setInternalValue(first);
  }, [isControlled, defaultValue, internalValue, tabOrderRef.current.join()]);

  const tabOrder = tabOrderRef.current;
  const ctx = useMemo(
    () =>
      ({
        baseId,
        orientation,
        value: selected,
        setValue,
        registerTab,
        unregisterTab,
        isDisabled: (v: string) => Boolean(disabledRef.current[v]),
        tabOrder,
      }) satisfies TabsContextValue,
    [baseId, orientation, selected, setValue, registerTab, unregisterTab, tabOrder.join("|")],
  );

  const rootOrientationClass =
    orientation === "vertical" ? "flex flex-row gap-4 items-start" : "flex flex-col";

  return (
    <TabsContext.Provider value={ctx}>
      <div className={`${rootOrientationClass} ${className}`.trim()}>{children}</div>
    </TabsContext.Provider>
  );
}

export type TabsListProps = {
  "aria-label"?: string;
  className?: string;
  children?: ReactNode;
};

export function TabsList({
  "aria-label": ariaLabel = "标签页",
  className = "",
  children,
}: TabsListProps) {
  const { orientation } = useTabsContext("TabsList");
  const flexDir =
    orientation === "vertical" ? "flex flex-col gap-1 shrink-0" : "flex flex-row gap-1 overflow-x-auto";
  return (
    <div
      role="tablist"
      aria-orientation={orientation}
      aria-label={ariaLabel}
      className={`${flexDir} ${className}`.trim()}
    >
      {children}
    </div>
  );
}

export type TabsTriggerProps = {
  value: string;
  disabled?: boolean;
  className?: string;
  children?: ReactNode;
};

function moveFocus(
  ctx: TabsContextValue,
  current: string,
  delta: number,
): void {
  const { tabOrder, isDisabled, setValue } = ctx;
  if (tabOrder.length === 0) return;
  let idx = tabOrder.indexOf(current);
  if (idx < 0) idx = delta > 0 ? -1 : tabOrder.length;
  for (let i = 0; i < tabOrder.length; i++) {
    idx = (idx + delta + tabOrder.length) % tabOrder.length;
    const v = tabOrder[idx];
    if (!isDisabled(v)) {
      setValue(v);
      document.getElementById(`${ctx.baseId}-tab-${v}`)?.focus();
      return;
    }
  }
}

function focusEdge(ctx: TabsContextValue, edge: "start" | "end") {
  const { tabOrder, isDisabled, setValue } = ctx;
  const seq = edge === "start" ? tabOrder : [...tabOrder].reverse();
  const v = seq.find((x) => !isDisabled(x));
  if (v !== undefined) {
    setValue(v);
    document.getElementById(`${ctx.baseId}-tab-${v}`)?.focus();
  }
}

export function TabsTrigger({
  value,
  disabled = false,
  className = "",
  children,
}: TabsTriggerProps) {
  const ctx = useTabsContext("TabsTrigger");
  const { baseId, orientation, value: selected, setValue, registerTab, unregisterTab } =
    ctx;

  useEffect(() => {
    registerTab(value, disabled);
    return () => unregisterTab(value);
  }, [value, disabled, registerTab, unregisterTab]);

  const selectedBool = selected === value;
  const base =
    "rounded-md px-3 py-2 text-sm font-medium transition-colors focus-visible:outline focus-visible:outline-2 focus-visible:outline-offset-2 focus-visible:outline-slate-400 disabled:pointer-events-none disabled:opacity-50";
  const inactive = "text-slate-600 hover:text-slate-900 hover:bg-slate-50";
  const active =
    "text-sky-800 bg-sky-50 ring-1 ring-inset ring-sky-200";
  const verticalMark =
    orientation === "vertical" && selectedBool
      ? "border-l-2 border-sky-600 pl-2"
      : orientation === "vertical"
        ? "border-l-2 border-transparent pl-2"
        : "";
  const horizontalMark =
    orientation === "horizontal" && selectedBool
      ? "border-b-2 border-sky-600 -mb-px"
      : orientation === "horizontal"
        ? "border-b-2 border-transparent -mb-px"
        : "";

  const onKeyDown = (e: KeyboardEvent<HTMLButtonElement>) => {
    if (disabled) return;
    const horizontalKeys = e.key === "ArrowLeft" || e.key === "ArrowRight";
    const verticalKeys = e.key === "ArrowUp" || e.key === "ArrowDown";
    if (orientation === "horizontal" && verticalKeys) return;
    if (orientation === "vertical" && horizontalKeys) return;

    if (orientation === "horizontal" && horizontalKeys) {
      e.preventDefault();
      moveFocus(ctx, value, e.key === "ArrowRight" ? 1 : -1);
      return;
    }
    if (orientation === "vertical" && verticalKeys) {
      e.preventDefault();
      moveFocus(ctx, value, e.key === "ArrowDown" ? 1 : -1);
      return;
    }
    if (e.key === "Home") {
      e.preventDefault();
      focusEdge(ctx, "start");
      return;
    }
    if (e.key === "End") {
      e.preventDefault();
      focusEdge(ctx, "end");
    }
  };

  return (
    <button
      type="button"
      role="tab"
      id={`${baseId}-tab-${value}`}
      aria-selected={selectedBool}
      tabIndex={selectedBool ? 0 : -1}
      disabled={disabled}
      className={`${base} ${inactive} ${selectedBool ? active : ""} ${horizontalMark} ${verticalMark} ${className}`.trim()}
      onClick={() => !disabled && setValue(value)}
      onFocus={() => !disabled && setValue(value)}
      onKeyDown={onKeyDown}
    >
      {children}
    </button>
  );
}

export type TabsPanelProps = {
  value: string;
  className?: string;
  children?: ReactNode;
};

export function TabsPanel({
  value,
  className = "",
  children,
}: TabsPanelProps) {
  const { baseId, value: selected } = useTabsContext("TabsPanel");
  const hidden = selected !== value;
  return (
    <div
      role="tabpanel"
      id={`${baseId}-panel-${value}`}
      aria-labelledby={`${baseId}-tab-${value}`}
      hidden={hidden}
      className={className}
    >
      {children}
    </div>
  );
}
```

**注意：** `useEffect` 初次选中依赖 `tabOrderRef.current.join()` 的 unstable trick —— 工程师实现时可改为 `registerTab` 内 `setOrderVersion(v => v+1)` 更干净；上述代码以「能跑通测试」为首要目标，合并时应替换 `rerender` 模式为显式 `orderVersion` state。
