# Select 组件 — 交互稿（交互 / 视觉 / 无障碍）

**组件**：表单单选下拉 Select  
**关联文档**：[PRD](./select-prd.md) · [开发者 API](./select-api.md)

---

## 1. 结构（信息架构）

```
┌─ 触发器（button，全宽）─────────────────────────────┐
│  当前选中 label 或 placeholder          [▼ 箭头]   │
└────────────────────────────────────────────────────┘
        │
        ▼ 展开（portal 否：跟随文档流 absolute）
┌─ 面板 ─────────────────────────────────────────────┐
│  [可选] 搜索框（type=search）                       │
├────────────────────────────────────────────────────┤
│  · 选项一                                           │
│  · 选项二（禁用）                                   │
│  · …                                                │
└────────────────────────────────────────────────────┘
```

- **触发器**：单行；左侧文案区；右侧 chevron；`aria-expanded` 反映展开，`aria-controls` 指向面板根节点 id。
- **面板**：位于触发器下方同一视觉列（`left/right` 贴齐触发器）；最大高度内滚动（当前实现 `max-h-72` + `overflow-auto`）。
- **列表**：纵向单列选项；加载中与选项列表互斥展示逻辑见 PRD / 实现。

---

## 2. 状态说明

| 状态 | 触发器 | 面板 | 搜索框 |
|------|--------|------|--------|
| 默认收起 | 可聚焦；显示选中 label 或 placeholder | 隐藏 | — |
| 展开（filterable） | `aria-expanded=true` | 可见 | 可见且打开后 **优先获焦** |
| 展开（非 filterable） | 同上 | 可见 | 隐藏；焦点进 **listbox** |
| 加载中 | 仍可关闭 | 列表区展示「加载中…」 | 可编辑（若存在） |
| 空列表 | 同展开 | 展示空占位（Vue 插槽 / React 属性） | 同左 |
| 整块 disabled | 不可点击；视觉弱化 | 不出现 | — |

---

## 3. 指针交互

| 操作 | 结果 |
|------|------|
| 点击触发器 | 切换展开 / 收起（未 disabled）。 |
| 点击面板外 | 收起；不改变选中（除非已完成一次选项点击）。 |
| 点击某选项 | 选中该项对应 value；收起；焦点回触发器。 |
| `mousedown` 在选项上 | `preventDefault` 避免焦点抢先丢失导致点击失效（实现约定）。 |
| Hover 选项 | 更新键盘高亮索引（与鼠标指向一致）。 |

---

## 4. 焦点顺序

1. Tab 进入页面时：触发器按文档顺序进入 Tab 顺序（面板关闭时不劫持 Tab）。
2. 打开面板：`filterable` → 焦点进搜索框；否则 → 焦点进 listbox（`tabindex=-1` 可编程聚焦）。
3. 选中或关闭：焦点回到 **触发器**。
4. `Escape`：关闭面板并焦点回触发器。

---

## 5. 键盘规范

### 5.1 触发器（焦点在 button）

| 按键 | 行为 |
|------|------|
| `Enter` / `Space` | 打开面板。 |
| `ArrowDown` | 打开面板。 |

（其它键交给浏览器默认；不要求在此模拟原生 `<select>` 的全部按键。）

### 5.2 面板内（事件挂在面板容器，输入框内按键向上冒泡）

| 按键 | 行为 |
|------|------|
| `ArrowDown` | 高亮移至下一个 **可用** 选项（跳过 disabled）。 |
| `ArrowUp` | 高亮移至上一个可用选项。 |
| `Home` | 高亮移至第一个可用选项。 |
| `End` | 高亮移至最后一个可用选项。 |
| `Enter` | 若有高亮项且可用 → 选中并关闭；否则无操作。 |
| `Escape` | 关闭面板，焦点回触发器。 |

高亮索引可为 `-1`（无高亮）；首次 `ArrowDown` 从第一个可用项开始。

## 6. 视觉规格（与实现一致）

| 元素 | 规范 |
|------|------|
| 主题色 | `sky` 强调选中 / 焦点环倾向；`slate` 边框与正文。 |
| 触发器 | 圆角 `rounded-lg`；边框 `border-slate-300`；hover 浅灰底。 |
| 选中项（列表内） | `font-medium` + `text-sky-800`。 |
| 键盘高亮 | `bg-sky-50`（与选中可同时存在时：二者样式可叠加）。 |
| 禁用选项 | 透明度降低；不可点；键盘跳过。 |
| Chevron | 展开时旋转 `180deg`。 |

完整类名以代码为准；**必须引入包内 `style.css`** 才能保证 Tailwind 编译进产物。

## 7. 无障碍（ARIA）

| 节点 | 约定 |
|------|------|
| 触发器 | `button`；`aria-haspopup="listbox"`；`aria-expanded`；`aria-controls`。 |
| 面板根 | `role="presentation"`；`tabIndex={-1}` 承接键盘。 |
| 搜索框 | `type="search"`；`aria-label="搜索选项"`（后续可 i18n）。 |
| 列表 | `role="listbox"`；`aria-labelledby` 指向触发器 id；`aria-busy` 反映加载；`aria-activedescendant` 指向当前高亮 option id。 |
| 选项 | `role="option"`；`aria-selected`；`aria-disabled`。 |

## 8. 文案默认值（可迭代）

| 场景 | 默认文案 |
|------|----------|
| 未选择 | `请选择`（`placeholder` 可覆盖） |
| 加载中 | `加载中…` |
| 空列表 | `暂无数据`（可被 Vue `#empty` / React `emptyContent` 替换） |
